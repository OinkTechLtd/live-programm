# live-programm 📺

Проект для автоматического поиска расписания телеканалов и просмотра эфира в стиле TikTok.

## Особенности
- **Поисковой робот:** Автоматический сбор расписания программ из открытых источников.
- **Интерфейс TikTok:** Вертикальный свайп каналов.
- **Плеер:** Интеграция `cdnplayerjs` для воспроизведения потоков.
- **Автообновление:** GitHub Actions обновляет расписание каждые 3 часа.

## Источники плейлистов
- `oinktechllc/livem3u`
- `oinktechltd/rulive` (запасной)

## Технологии
- Python (для робота-парсера)
- HTML/JS/CSS (Frontend)
- GitHub Actions (Automation)
